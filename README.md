# Digital Platform for Tracking Vaccination Coverage

### Overview
The Digital Platform for Tracking Vaccination Coverage is a comprehensive, full-stack web application designed to streamline the tracking, administration, and reporting of public health vaccination efforts. Built to handle relational data securely, the platform bridges the gap between patient record management and high-level administrative oversight.


### Key Features
Centralized Patient Database: Securely maintains and manages records of vaccinated individuals, tracking critical data such as vaccine type, administered doses, and vaccination center locations.

Automated Certificate Generation: Dynamically generates and issues downloadable PDF vaccination certificates for individuals once their required doses are completed.

Admin Analytics Dashboard: Provides administrators with a real-time, interactive heatmap and data visualization dashboard to monitor geographical vaccination coverage and identify regions needing more resources.

### Tech Stack
Frontend: React.js, Tailwind CSS
Backend: Node.js, Express.js
Database: PostgreSQL
